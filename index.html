利用空閒之餘開發專案 (做中學成果)

1. FastAPI 後端分層架構與驗證系統 (FastAPITest)

為了掌握現代後端架構與測試思維，我建立了一個 FastAPI 後端開發與練習專案：

  • 架構分層設計：實作 Router ➔ Service ➔ Repository ➔ Database 分層，降低耦合度，提升系統可維護性。
  • 安全驗證機制：使用 JWT (JSON Web Token) 進行登入認證，並透過 bcrypt 雜湊演算法保護使用者密碼。
  • 測試與環境配置：使用 pytest 撰寫單元測試，並運用相依性注入 (DI) 抽換為 FakeRepository，實現輕量且獨立的測試環境。
  • 資料庫與容器化：以 SQLAlchemy ORM 進行資料存取，支援文件 CRUD（新增、查詢、部分更新、刪除）操作，並使用 Docker-Compose 整合 PostgreSQL 資料庫服務。
